⚡ QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: Blank page / Nothing showing?
Solution: Press Ctrl + F5 (Hard refresh) or Ctrl + Shift + Delete (Clear cache)

Problem: Styles not loading?
Solution: Make sure style.css is in the same folder as index.html

Problem: JavaScript not working?
Solution: Open browser console (F12) and check for errors

Problem: Live Server not working?
Solution: Reinstall Live Server extension from VS Code

Problem: Services list empty?
Solution: Check script.js and make sure data is loaded
