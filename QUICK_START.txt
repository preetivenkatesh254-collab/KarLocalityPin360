❓ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLEM: Website won't load
SOLUTION: 
   • Refresh browser (Ctrl + F5)
   • Hard refresh (Ctrl + Shift + Delete)
   • Check Live Server is running

PROBLEM: Styles look weird
SOLUTION:
   • Make sure style.css is in same folder as index.html
   • Refresh browser (Ctrl + F5)
   • Check browser console (F12)

PROBLEM: Services list is empty
SOLUTION:
   • Check script.js has data
   • Open browser console (F12)
   • Look for JavaScript errors

PROBLEM: Live Server not working
SOLUTION:
   • Uninstall and reinstall Live Server
   • Use Python method instead
   • Close and reopen VS Code

PROBLEM: Port already in use
SOLUTION:
   • Use different port number
   • Or close the other application using that port
